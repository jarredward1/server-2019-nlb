## Introduction
This homelab project focused on deploying a Network Load Balancer (NLB) using Windows Server 2019 to optimize traffic distribution across multiple servers. I configured the NLB feature alongside IIS on virtual machines, ensuring seamless integration and high availability. By setting up static IPs, clustering servers, and testing web traffic distribution, I validated the system's performance and reliability.
